# Housing Prices Dataset Analysis

This repository contains my assignment files for the Housing Prices Dataset analysis.

## Files included
- `housing_price_analysis.ipynb`: Colab analysis notebook
- `presentation/housing_price_presentation.pptx` or `housing_price_presentation.pptx`: presentation slides
- `screenshots/`: screenshots used in the presentation

## Project summary
This project analyzes the Housing Prices Dataset from Kaggle.
The analysis explores how factors such as area, bedrooms, and preferred area affect housing prices.
